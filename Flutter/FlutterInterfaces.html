Getting Started with Flutter: Building User Interfaces

Published 29 Feb 2024

üëã In Flutter, creating visually appealing and responsive user interfaces is made possible through various layout options. This guide provides an overview of the fundamental layout widgets available in Flutter, including rows, columns, and containers, along with examples and explanations for each.

Rows

Rows are horizontal arrangements of widgets in Flutter, where children are laid out in a horizontal line.

  • The Row widget arranges its children widgets horizontally, from left to right.
  • In the example, two Text widgets ('Hello' and 'World') are placed side by side within the row.
				
Row(
	children: [
		Text('Hello'),
		Text('World'),
	],
	)

Loops (for, while, do-while)

Loops are used to execute a block of code repeatedly until a specified condition is no longer true. Dart supports three types of loops: for, while, and do-while.

  • A for loop is typically used when you know how many times you want to execute a block of code.
  • A while loop continues to execute a block of code as long as the specified condition remains true.
  • A do-while loop is similar to a while loop, except that it always executes the block of code at least once before checking the condition.

For Loop

				
void main() {
	for (int i = 1; i <= 5; i++) {
		print('Number: $i');
	}
	}

While Loop

				
void main() {
	int count = 1;
	
	while (count <= 5) {
		print('Count: $count');
		count++;
	}
	}

Do-While Loop

				
void main() {
	int count = 1;
	
	do {
		print('Count: $count');
		count++;
	} while (count <= 5);
	}

Functions

Functions are reusable blocks of code that perform specific tasks. They help organize code, promote reusability, and make it easier to maintain and debug programs.

  • Functions in Dart consist of a function signature (name, parameters) and a function body (block of code).
  • Parameters are variables passed to the function to provide input values.
  • The return type specifies the type of value that the function will return.

Function without parameters

				
void greet() {
	print('Hello, World!');
	}
	
	void main() {
	greet(); // Call the function
	}

Function with parameters and return value

				
int add(int a, int b) {
	return a + b;
	}
	
	void main() {
	int result = add(5, 3);
	print('Result: $result');
	}
